# antd-img-crop-resize

> A react library that take img uploaded by antd upload component, crop and resize it

[![NPM](https://img.shields.io/npm/v/antd-img-crop-resize.svg)](https://www.npmjs.com/package/antd-img-crop-resize) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save antd-img-crop-resize
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'antd-img-crop-resize'
import 'antd-img-crop-resize/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
